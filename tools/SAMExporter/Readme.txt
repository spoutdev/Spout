#########################

Author: Karang
Version: v1.0

/!\ This is still in developpement, I only released it because Roy asked to :D

HOW TO INSTALL:
Drag the io_anim_sam into your blender's addon folder.
Restart blender.
Go into File>User preferences
Go in the tab Addon
Find the plugin then tick the box to enable it.

HOW TO USE:
Select you mesh (not the armature)
Then go into File>Export>Spout Animation Model (SAM)
